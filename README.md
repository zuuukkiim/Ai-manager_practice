# 🥕 AI Manager (AI 대화 매니저)

다양한 AI 툴과의 대화 내역을 한곳에서 관리하고, 위젯으로 빠르게 확인하는 iOS/macOS 앱입니다.

## ✨ 주요 기능
- **Dashboard**: 엑셀 스타일의 정돈된 인터페이스로 AI 툴별 대화 요약 확인
- **Apple Standard UI**: SF Symbols를 활용한 직관적인 하단 탭 바 메뉴
- **Widget**: 아이폰/맥북 홈 화면에서 실시간 AI 대화 주제 확인

## 🛠 Tech Stack
- **Language**: Swift
- **Framework**: SwiftUI, WidgetKit
